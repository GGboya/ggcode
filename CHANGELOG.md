## 版本 1.0.0 - 2025-06-15

### 🚀 初始版本发布

#### 核心功能
- **艾宾浩斯学习算法**
  - 完整复习路径：7次复习（不会做的题目）
  - 简化复习路径：3次复习（独立AC的题目）
  - 智能复习时间安排

- **用户系统**
  - 用户注册/登录
  - JWT 身份认证
  - 个人学习数据管理

- **学习管理**
  - 学习计划创建和管理
  - 每日学习任务推送
  - 学习进度统计
  - 自动打卡系统

- **题库系统**
  - 官方 LeetCode Hot 100 题库
  - 自定义题库创建
  - 题目管理功能

#### 技术架构
- 后端：Go + Gin + GORM + MySQL
- 前端：HTML/CSS/JavaScript + Bootstrap 5
- 数据库：MySQL（支持 SQLite 迁移）
- 部署：Docker 支持

## 🔄 升级指南

### 从 SQLite 迁移到 MySQL
```bash
# 1. 导出 SQLite 数据
sqlite3 ggcode.db .dump > ggcode_backup.sql

# 2. 使用自动部署脚本
./deploy_mysql.sh
```

### 环境配置更新
```env
# 新增 MySQL 配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=ggcode
```
