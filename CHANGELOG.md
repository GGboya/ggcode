## 🔄 升级指南

### 从 SQLite 迁移到 MySQL
```bash
# 1. 导出 SQLite 数据
sqlite3 ggcode.db .dump > ggcode_backup.sql

# 2. 使用自动部署脚本
./deploy_mysql.sh
```

### 环境配置更新
```env
# 新增 MySQL 配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=ggcode
```
