# GGCode - 智能算法学习平台

## 🎯 项目简介

GGCode 是基于艾宾浩斯遗忘曲线理论的智能算法学习平台，帮助程序员科学高效地掌握算法题目。

## ✨ 核心功能

### 🔐 用户系统
- 用户注册/登录，JWT身份认证

### 📚 题库管理
- **官方题库**：LeetCode Hot 100 精选题目
- **个人题库**：创建、编辑、删除自定义题库
- **共享题库**：Star、Fork 优质题库，社区共享

### 🧠 智能学习算法
- **艾宾浩斯遗忘曲线**：科学安排复习时间
- **两种学习模式**：
  - 独立AC：第一次遇到题目就独立AC，后续不再复习; 不会做之后再独立 AC，按照预期间隔复习
  - 不会做：7次复习（1天→2天→4天→7天→15天→30天→60天）
- **个性化推荐**：每日智能推送学习任务

### 📊 学习统计
- 学习进度可视化
- 掌握率统计
- 自动打卡系统
- 连续学习天数追踪

### ⚡ 在线评测机
- **多语言支持**：C++、Java、Python、Go
- **安全沙箱**：Linux 命名空间隔离 + cgroup 资源限制
- **实时判题**：基于 go-judge 的高性能评测引擎
- **智能检测**：时间超限、内存超限、答案错误等状态判定
- **灵活配置**：支持自定义测试用例和运行参数

## 🚀 技术栈

- **后端**：Go + Gin + GORM + MySQL
- **前端**：HTML/CSS/JavaScript + Bootstrap 5
- **认证**：JWT Token
- **部署**：Docker 支持
